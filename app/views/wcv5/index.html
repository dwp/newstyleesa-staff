Work Coach view – version 5

Changes in this version

  • Claim actions are now shown on the left of the screen
  • There’s now a ‘Claim overview’ made up of a claim summary and the timeline (blue header has less information in it)
  • ‘Withdraw this claim’ has its own link in the sidebar
  • Exploring whether the agent UI can replace the Agent Lead Porcess sheet (ALP), so adopted more of a walkthrough approach to changing the claim status
  • A buffer screen informs users what will happen before they change the status
  • ‘You cannot undo this action’ messages now appear above the button
  • There’s a ’Back to their claim’ link status confirmation screens

Screens

  • Appointment booked
  • Failed to attend
  • Verified
  • New appointment needed
  • Withdrawn
  • Processed
  • Disallowed
